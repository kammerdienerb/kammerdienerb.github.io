I am a performance engineer working on Brendan Gregg's team at Intel. We develop advanced performance analysis tools used to solve performance problems in all layers of the software stack.

Before joining Intel, I graduated with my Ph.D. in Computer Science at the University of Tennessee, Knoxville. While there, my focus was in developing low-level software systems aiming to improve the memory behavior of applications in more complex memory architectures.

I also work on several side projects in the areas of developer tools, programming language design, compilers, and operating systems.
